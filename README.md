# 🛡️ SafeVoice – Women Safety & Support Platform

SafeVoice is a MERN stack web application built to support women by guiding them to **official government complaint portals**, connecting them with **NGOs**, spreading **legal awareness**, and collecting **anonymous analytics** — all while strictly respecting privacy.

This project focuses on **real-world impact**, not just features.

---

## 🌸 Why SafeVoice?

Many women hesitate to file complaints due to fear, confusion, or lack of guidance.  
SafeVoice does **not replace government systems** — instead, it helps users reach the **right place**, with the **right information**, safely.

---

## ✨ Key Features

### 📝 Complaint Guidance
- Categorized issue selection (Cyber Abuse, Domestic Violence, Workplace Harassment, etc.)
- Direct redirection to **official Indian government portals
